# FundBridge 🌉

> **Empower Micro-Entrepreneurs. Build Better Futures.**

FundBridge is a comprehensive, dual-platform ecosystem designed to drive economic growth and social change. It bridges the gap between impactful investors and promising micro-entrepreneurs by providing accessible tools for funding, tracking, and enterprise management.

## 🚀 Project Overview

The ecosystem is divided into two distinct platforms tailored for our primary users:

### 💻 Web Platform (For Investors)
A dedicated portal for investors to discover, fund, and track micro-enterprises.
* **Browse Enterprises:** Explore a curated, vetted list of micro-entrepreneurs across various industries.
* **Make Impactful Investments:** Securely choose investment amounts and complete the funding process with ease.
* **Track Progress:** Monitor your active investment portfolio and watch the enterprises grow over time through a personalized dashboard.

### 📱 Mobile Application (For Entrepreneurs)
A streamlined, accessible mobile application designed specifically for micro-entrepreneurs.
* **Easy Onboarding:** Quickly create a profile and list an enterprise for funding.
* **Request Capital:** Submit funding goals and business plans to the investor network.
* **Manage Growth:** Update investors on milestones, track received funds, and manage the enterprise directly from a smartphone.

---

## 🛠️ Tech Stack

**Frontend (Web - Investor Portal):**
* React.js
* Tailwind CSS
* Redux (State Management)

**Frontend (Mobile - Entrepreneur App):**
* React Native

**Backend & Database:**
* Node.js with Express.js
* MongoDB (Mongoose)
* RESTful APIs (or GraphQL)

## ⚙️ Local Setup & Installation

To get this project up and running on your local machine, follow these steps:

### Prerequisites
* Node.js (v16 or higher)
* MongoDB (Local instance or MongoDB Atlas URI)
* React Native development environment (Android Studio / Xcode)
